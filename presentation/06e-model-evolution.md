# Model Evolution & Future

## Timeline of Computer Use AI

**2023 - Early Vision Models**
- GPT-4 Vision launched
- Can describe images
- No action capabilities

**2024 Q3 - Anthropic Breakthrough**
- Claude 3.5 with Computer Use (October 2024)
- First production computer use API
- `computer_20241022` tool

**2024 Q4 - Microsoft Enters**
- Azure AI `computer-use-preview` launched
- Based on OpenAI partnership
- Integrated into Azure AI Foundry

**2025 - Current State**
- Multiple providers supporting computer use
- Improved accuracy and speed
- Enterprise adoption beginning
